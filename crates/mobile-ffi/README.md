# LocalGPT Mobile FFI

Mobile bindings for embedding LocalGPT core into iOS and Android applications. This crate uses [UniFFI](https://github.com/mozilla/uniffi-rs) to generate foreign language bindings (Swift, Kotlin).

## Features

- **Multi-Platform Support**: Static and dynamic libraries for iOS and Android.
- **Foreign Language Bindings**: Automated generation of Swift and Kotlin interfaces.
- **Embedded Core**: Full access to LocalGPT reasoning and memory features within mobile apps.
