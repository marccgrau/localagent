# LocalGPT Gen

LocalGPT Gen is a specialized tool for AI-driven 3D scene generation, built with the Bevy engine. It allows LocalGPT to create, manipulate, and export explorable 3D environments.

## Features

- **Bevy Engine Integration**: High-performance rendering and physics.
- **AI-Driven Scene Generation**: Geometry, materials, and lighting controlled by LocalGPT.
- **glTF/GLB Export**: Support for exporting generated scenes for use in other 3D applications.
- **Procedural Audio**: Dynamic environmental soundscapes powered by `fundsp`.
